## Commands
| Command | Description | Admin only |
|---------|-------------|------------|
| /StockMarket ManagementGUI             | Open the management GUI to create/remove/change markets    | :heavy_check_mark: |
| /StockMarket save                      | Save the Stockmarket data                                  | :heavy_check_mark: |
| /StockMarket load                      | Load the Stockamrket data                                  | :heavy_check_mark: |
| /StockMarket closeAllMarkets           | Disables trading for all players on all markets            | :heavy_check_mark: |
| /StockMarket openAllMarkets            | Enables trading for all players on all markets             | :heavy_check_mark: |

## ManagementGUI
The most important window for admins is the `ManagementGUI`. <br>
Use the following command to open it: `/StockMarket ManagementGUI`<br>
That window lets you manage the whole stock market.
